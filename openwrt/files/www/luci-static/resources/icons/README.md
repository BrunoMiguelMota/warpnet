# WarpNET Logo Files

This directory should contain the WarpNET logo files for the web interface.

## Required Files
- `warpnet-logo.png` - Main logo file

## Download Script
Run the logo download script to automatically download and optimize the logo:

```bash
./scripts/download-logo.sh
```

## Manual Download
If the script doesn't work, manually download:
- URL: https://warpnet.es/images/logowarpnet.png
- Save as: `warpnet-logo.png` in this directory

## Logo Requirements
- Format: PNG
- Transparent background recommended
- Size: 64x64 to 256x256 pixels
- Optimized for web use
